CAN

Overview

The Basic Extended CAN peripheral, interfaces the CAN network. It supports the CAN protocols version 2.0 A and B. It has been designed to manage a high number of incoming messages efficiently with a minimum CPU load. It also meets the priority requirements for transmit messages.
